# 📝 TOC Notemaker

**TOC Notemaker** is a sleek, professional web application designed for Teachers On Call (TOCs) and Substitute Teachers to create beautiful, structured, and easy-to-read lesson reports. No more messy handwritten notes—generate a polished PDF that your colleagues will love.

## ✨ Features

- **🚀 Instant PDF Generation:** Create professional reports in seconds.
- **📅 Dynamic Schedules:** Built-in support for regular and collaboration (Wednesday) schedules with automatic block rotations.
- **📸 Image Support:** Attach photos of student work or classroom activities directly into your notes with captions.
- **🎨 Live Preview:** See exactly how your report looks as you type.
- **📱 Mobile Optimized:** Write your notes on the go using your phone or tablet.
- **⚖️ Adjustable Layout:** Fine-tune the "stretch" and "squish" of your report to fit more or less content per page.
- **🌓 Modern UI:** A beautiful dark-themed interface built with a focus on user experience.

## 🚀 Getting Started

TOC Notemaker is a **zero-install, static web application**. You don't need to install anything!

### 1. Run Locally
Simply clone the repository and open `index.html` in any modern web browser:

```bash
git clone https://github.com/your-username/toc-notes.git
cd toc-notes
open index.html
```

### 2. Use it Anywhere
Since it's a single-file application (powered by CDNs), you can host it on **GitHub Pages**, **Vercel**, or even just keep the HTML file on your desktop.

## 🛠 How to Use

1.  **Set the Stage:** Enter your name and select the date and type of absence (Full Day, AM, PM, or Multi-Day).
2.  **Fill the Blocks:** For each class period, enter the Course Name, Room Number, and your notes.
3.  **Add Visuals:** Click the image icon to upload photos. You can add captions and adjust the size (Small, Medium, or Large) to fit the layout.
4.  **Review:** Use the **Live Preview** on the right (or the preview button on mobile) to check your work.
5.  **Download:** Click **Download PDF** to save your professional report.
